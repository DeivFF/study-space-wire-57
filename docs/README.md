### 🚀 Começando

Se você é novo no projeto, recomendamos ler a documentação na seguinte ordem:

1. Comece com **Configuração e Setup** para preparar seu ambiente
2. Leia **Arquitetura do Projeto** para entender a estrutura geral
3. Consulte **Padrões de Desenvolvimento** antes de contribuir com código
4. Use **Componentes e UI** e **API e Backend** como referência durante o desenvolvimento

### 🎯 Objetivo do Projeto

O Study Space é uma plataforma educacional que permite:

- Autenticação segura de usuários
- Feed social para interação entre estudantes
- Sistema de amizades e conexões
- Notificações em tempo real
- Interface responsiva e moderna
- Banco de questões
